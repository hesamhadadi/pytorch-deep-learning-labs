# PyTorch Deep Learning Labs

This repository contains my weekly deep learning laboratory exercises using PyTorch.

The labs cover fundamental and advanced topics in deep learning including:

- PyTorch tensors
- Autograd
- Neural networks
- CNNs
- Data loading with torchvision
- Image datasets (TinyImageNet)
- Model training pipelines

Each folder corresponds to a weekly lab session.

## Technologies
- Python
- PyTorch
- Torchvision
- NumPy
- Matplotlib
- Google Colab

## Dataset
Some labs use the Tiny ImageNet dataset:
http://cs231n.stanford.edu/tiny-imagenet-200.zip
